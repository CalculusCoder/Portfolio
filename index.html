Smart Brain

Full stack web application that detects faces given an image URL. Smart Brain is connected to a machine learning REST API that calculates the face location in images. Front-end was built using React.js, HTML, CSS, and Javascript. The back-end server was created and connected to the front-end using Node.js and Express.js. PostgreSQL was used as the relational database, and was connected to the server using Knex.js.
